Si existe una función getIcon() u similar que recibe nombres de
iconos desde una API o base de datos, los nombres llegarán en
PascalCase (ej: "Globe", "FileCode2", "LayoutTemplate") porque
así los guarda lucide-svelte en la DB.

DEBES implementar esta lógica de conversión automática:

  function getIcon(iconName: string): string {
    // 1. Convertir PascalCase → kebab-case
    const kebab = iconName
      .replace(/([A-Z])/g, (match, letter, offset) =>
        offset > 0 ? '-' + letter.toLowerCase() : letter.toLowerCase()
      )
      .replace(/(\d+)/g, '-$1')
      .replace(/--+/g, '-')
      .replace(/-+$/, '');

    // 2. Aplicar renombrados de Lucide v0.468+
    const renames: Record<string, string> = {
      'globe': 'earth',
      'globe-icon': 'earth',
    };

    const resolved = renames[kebab] || kebab;

    // 3. Retornar con prefijo lucide:
    //    Si no existe el icono, Iconify mostrará vacío en lugar de crash
    return `lucide:${resolved}`;
  }
